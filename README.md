# 📚 Minjoo’s Shelf

감성 책 추천 웹사이트입니다.  
노예 1 (Claude), 노예 2 (GPT), 노예 3 (제) 함께 만든 책 추천 프로젝트!

🔥 실제 배포 링크:  
🔗 https://minjooshelf.vercel.app/

---

## 🚀 프로젝트 소개

**Minjoo’s Shelf**는 감성적이고 직관적인 UI로 책을 추천하고, 사용자가 좋아할 만한 책을 쉽게 찾을 수 있도록 도와주는 개인 책장 웹 애플리케이션입니다.  
다양한 책들을 둘러보고 나만의 책 리스트를 만들어 보세요.

---

## 💡 주요 기능

- 📘 책 검색 및 추천 기능
- ⭐ 사용자 감성 기반 책 정보 제공
- 📚 마이 셸프(Book Shelf) 스타일 UI 구성
- 📍 반응형 웹 지원

---

## 🧠 기술 스택

프로젝트는 아래 기술들을 기반으로 만들어졌어요:

- ⚛️ **Next.js** — React 기반의 프레임워크
- 🛠 Typescript — 타입 안전성 확보
- 💅 CSS — 스타일링
- ☁️ Vercel — 손쉬운 배포 및 운영

---

## 📦 설치 및 실행 방법

로컬에서 직접 실행해 보고 싶다면 아래 단계를 따라주세요.

```bash
# 레포지토리 클론
git clone https://github.com/MINJOOOONG/Minjoo-s-shelf.git

# 프로젝트 폴더로 이동
cd Minjoo-s-shelf

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
