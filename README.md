## 🚀 프로젝트 소개

**Minjoo’s Shelf**는  
“책을 고르는 과정이 어렵다”는 문제에서 출발한 프로젝트입니다.

몇 가지 질문만으로 사용자의 독서 성향을 분석하고,  
그 결과를 바탕으로 책을 추천해 주는 **가볍지만 명확한 UX**를 목표로 구현했습니다.

개인 프로젝트이지만,  
실제 서비스 흐름을 고려해 **기획 → 구현 → 배포**까지 전 과정을 경험했습니다.

---

## ✨ 주요 기능

- 🧪 **독서 성향 테스트**
  - 간단한 질문을 통해 사용자 유형 분류

- 📚 **유형 기반 책 추천**
  - 테스트 결과에 따라 추천 도서 노출

- 🔊 **TTS(Text-to-Speech) 결과 안내**
  - 추천 페이지 진입 시  
    “당신의 독서 유형은 ○○입니다” 음성 자동 재생

- 🛒 **외부 서점 연동**
  - 도서 구매 버튼 클릭 시  
    교보문고 / 알라딘 / YES24로 이동

- 📱 **반응형 UI**
  - 모바일 / 데스크톱 환경 대응
